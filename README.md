# TIC-TAC-TOE Game
- Pick the value **X** or **O** you want to play with
- Choose the board size:
  - 3x3
  - 4x4
  - 5x5
  - 6x6
  - 7x7
  - 8x8
- Decorate your cells with any colors you want
